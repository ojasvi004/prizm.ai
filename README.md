<p align="left">
 Prizm AI is a <b>deliberative multi-agent platform</b> that uses structured debate and <b>real-time fact-checking</b> to turn complex reports into actionable insights. It transforms static data into transparent, multi-perspective intelligence by pitting specialized AI agents against each other to ensure every conclusion is rigorously tested.
</p>

## âœ¨ Features

| Feature                          | Description                                                                 |
| -------------------------------- | --------------------------------------------------------------------------- |
| **Multimodal Extraction** | Ingests **text, images, and charts** to identify core factors for analysis. |
| **Direct Agent-to-Agent Debate** | Facilitates **structured conflict** where agents challenge each other's logic.|
| **Real-Time Fact Verification** | Uses **SERP API tools** to ground claims with live, external web data.      |
| **A2A Coordination Flow** | Governs the **sequence of context exchange** and turn-taking between agents.|
| **Multi-Agent Synthesis** | Merges **opposing views** into a single, unified, and actionable report.    |
| **Transparent Reasoning Logs** | Exposes **full debate traces** and execution-flow visualizations for trust. |
| **Parallel Processing Hub** | Executes **verification and reasoning** simultaneously to reduce latency.   |
| **Decision-Making Verifier** | Ensures conclusions are based on **evidence** rather than model bias.       |
| **Structured Output Schemas** | Uses **strictly typed handshakes** to prevent data loss or hallucination.   |
| **Actionable PDF Reporting** | Automatically converts deliberation into a **professional document**.       |
